## [1.0.0] - 2025-01-17

### Added
- 구글 시트 연동 기능
- 엑셀 업로드 및 데이터 붙여넣기 기능
- 기준 날짜 선택 (주간 단위, 월요일 시작)
- 시트별 탭 관리 기능
- "추출테스트" 시트 G열 자동 수식 추가 (C+D)
- 전체 선택/해제 버튼
- 시트 목록 가져오기 API
- 데이터 추가 API
- 수식 업데이트 API

### Features
- 다중 시트 선택 및 관리
- 데이터 필터링 (주간 기준)
- 구글 시트 자동 데이터 추가
- 반응형 UI 디자인
